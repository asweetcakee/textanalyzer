Text Analysis Application
Overview
The Text Analysis Application automates text analysis using Natural Language Processing (NLP) techniques. It enables users to efficiently process text documents, extract keywords, and generate summaries. The application supports DOCX and PDF formats, providing a versatile tool for text data analysis.

Features
Automated text analysis using NLP methods.
Extraction of keywords and generation of concise summaries.
Support for DOCX and PDF document formats.
User-friendly interface for easy document handling.
